How to Install
....................................................
1. Create dev environment
    Bash: python -m venv {path} or
    Use your IDE to create this for you 
2.Copy the repository to your computer, you should have received a GitHub invitation from me.
    Bash:  git clone https://github.com/reubenmcc/TechForum2026.git
3.Install packages, run the following commands...
    cd {path_to_your_enviorment}
    cd Scripts
    activate
    cd..
    pip install -r requirements.txt
4. Enter in key api
    In the environment create a text file called keys.env
    Add the following lines to keys.env: ANTHROPIC_API_KEY = {API Key}
5. Click on run.bat
